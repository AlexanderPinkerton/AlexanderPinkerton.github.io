Void

Void is an encrypted chat platform that aims to provide a clean and responsive webservice for users across all platforms. Void uses 256-bit AES encryption standard with cipher block chaining to ensure that your messages can never be read by unauthorized users. Due to the encryption being performed on the client-side, the Void servers are never able to read users data and it is never stored in a database. The Void simply passes messages between users and provides a client which can ensure that each user can read the messages. The Void works on the assumption that its users are doing their part in ensuring that their passwords are very secure and that nobody unwanted knows them. As long as the user does their part, the Void ensures secure communication of messages between users from varying platforms.

  • Date: August 2016-Current

Machine Learning Research

Working under the universities leading machine learning professor I spent the summer of 2016 working with convolutional neural networks to perform computer vision tasks. Fine-tuned an existing model to perform regression on transient attributes in a landscape image, specifically cloudiness. Implemented a custom loss layer in python to change the existing classification architecture into one that supported regression. As convolutional neural networks are an end-to-end system, the input was an image and the output was a cloudiness percentage.

  • Date: May-July 2016
